# Extension Icons

Place your extension icons here:
- icon16.png (16x16)
- icon48.png (48x48)
- icon128.png (128x128)

You can create these icons using any image editor or online icon generator.

For now, you can use placeholder icons or create them with tools like:
- Canva
- Figma
- Adobe Illustrator
- Or use emoji-to-icon generators online

Recommended design: A briefcase or document icon with blue color scheme (#0066cc)
